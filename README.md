ENG : HA Intelligence — HACS Integration
Connect your Home Assistant to the HA Intelligence backend.

Installation
Open HACS in Home Assistant
Go to Integrations → Explore → search for "HA Intelligence"
Install and restart Home Assistant
Go to Settings → Integrations → Add → HA Intelligence
Paste your API key from the dashboard at https://ha-intelligence.netlify.app




ITA : # HA Intelligence — Integrazione HACS

Collega il tuo Home Assistant al backend HA Intelligence.

## Installazione
1. Apri HACS in Home Assistant
2. Vai su Integrazioni → Esplora → cerca "HA Intelligence"
3. Installa e riavvia Home Assistant
4. Vai in Impostazioni → Integrazioni → Aggiungi → HA Intelligence
5. Incolla la tua API Key dal pannello su https://ha-intelligence.netlify.app
